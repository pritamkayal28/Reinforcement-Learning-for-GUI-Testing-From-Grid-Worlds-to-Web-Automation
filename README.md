This project presents a Reinforcement Learning (RL)-based intelligent system designed for dynamic grid navigation and transfer learning applications. The system trains an RL agent in a custom GUI grid environment where the agent learns optimal paths, obstacle avoidance, and reward maximization through continuous interaction with the environment. The project implements Q-Learning techniques to improve decision-making and adaptive learning performance.

The model is evaluated using various performance metrics such as average reward, state coverage, cumulative reward, and moving average reward. In addition, the project explores transfer learning by transferring the knowledge learned in one grid environment to a different environment, reducing training time and improving learning efficiency compared to training from scratch.

The project also demonstrates the application of RL in web-based environments by simulating automated interaction tasks such as login page navigation and action prediction. Visualization techniques including reward graphs, heatmaps, and training comparison plots are used to analyze agent performance. The overall objective of this project is to develop an intelligent and adaptive learning system capable of solving dynamic navigation and automation problems efficiently using Reinforcement Learning and transfer learning concepts.
